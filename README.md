## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Development

```bash
# Start dev server
npm run dev

# Run linter
npm run lint

# Type checking
npm run typecheck
```

Visit `http://localhost:4321` to see the application.
